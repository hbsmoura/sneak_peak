Olá, visitante! Seja bem vind@ ao Skeak Peek.

Em tradução livre, "sneak peek" significa uma olhadinha furtiva, uma espiada. A expressão é bastante utilizada para se referir a trailers/teasers de produções cinematográficas. Sneak Peek, para nós, é o resultado de um desafio guiado, proposto pelo desenvolvedor/instrutor Felipe Aguiar (Link para o Github - Link para o LinkedIn), dentre as atividades das trilhas da Digital Innovation One. O sistema tem como principal, senão único objetivo, por em prática os conhecimentos obtidos por quem o desenvolve. Embora cada passo inicial tem sido guiado, o sistema de certa forma extende a proposta inicial, como uma classe herdeira, tentando abranger algumas habilidades em desenvolvimento, sem, contudo, fugir da proposta de um sistema em página única (Single Page Application), rodando totalmente no navegador (User Side), construído com base em HTML5, CSS3 e Javascript ES6.
